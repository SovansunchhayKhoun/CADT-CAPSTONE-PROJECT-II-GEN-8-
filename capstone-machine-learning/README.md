### Prerequisites

- Python (>=3.x)
- Jetbrains PyCharm or DataSpell (IDE)

### Installation

Clone the repository and install dependencies:

```sh
pip install -r requirements.txt
```

### Run the project

Make sure that this command is run in the `src` directory

```sh
$ uvicorn api:app --reload
```
